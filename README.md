# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project performs customer segmentation using K-Means clustering based on Annual Income and Spending Score.

## 📊 Techniques Used
- Exploratory Data Analysis (EDA)
- Statistics (mean, distribution analysis)
- K-Means Clustering
- Elbow Method for optimal clusters

## 📂 Dataset
Mall Customers Dataset

## 📈 Results
Identified 5 customer groups:
- Low income, low spending
- High income, high spending (VIP customers)
- Low income, high spending, etc.

## 🧠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 🚀 Outcome
Helps businesses understand customer behavior and improve marketing strategies.
